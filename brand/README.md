# IntentBid Brand Assets

Welcome to the IntentBid brand folder. 

## Contents
* `brand-guidelines.md` - The source of truth for IntentBid's positioning, messaging, visual aesthetic, and voice.
* `logos/` - A collection of 3 newly designed, high-resolution SVG logos embodying the new "Copilot" aesthetic.

### Logo Concepts
1. **The Prism Engine (`intentbid-logo-1-prism.svg`)**: Geometric layers representing the 6-layer Intent Framework locking together to create a proposal.
2. **The AI Monogram (`intentbid-logo-2-monogram.svg`)**: A sleek and modern interconnected "I" and "B", ideal for profile pictures, favicons, and social avatars. 
3. **The Upward Bid (`intentbid-logo-3-chevron.svg`)**: A stylized double-arrow or chevron pointing forward/upward, representing velocity (10x proposal speed) and winning momentum.

To preview the logos, simply open the `.svg` files in any modern web browser or design tool like Figma.
