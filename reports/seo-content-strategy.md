## Blog Technical Setup

### URL Structure
```
/blog                          # Blog index
/blog/ai-proposal-writing      # Article (no dates)
/blog/category/guides          # Category pages
/blog/tag/rfp                  # Tag pages
```

### File Structure
```
src/app/(public)/blog/
├── page.tsx                   # Blog index
├── [slug]/
│   └── page.tsx              # Article template
├── category/
│   └── [category]/
│       └── page.tsx          # Category page
└── posts/
    └── *.mdx                 # MDX content files
```

### MDX Frontmatter
```yaml
---
title: "Article Title"
description: "Meta description"
date: "2026-01-30"
author: "Author Name"
category: "guides"
tags: ["rfp", "ai", "proposals"]
image: "/blog/images/article-og.png"
---
```
