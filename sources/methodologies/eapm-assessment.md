## Overview

**eAPM** (Economic Application Portfolio Management), now branded as **Clear Sight IT Decision Maker**, is Capgemini's proprietary assessment tool and methodology for comprehensive application portfolio analysis and cloud migration planning.

## Core Capabilities

### Portfolio Assessment
- In-depth view of entire IT portfolio
- Cloud readiness scoring per application
- Migration effort calculation for all scenarios
- Blocker and impediment identification

### Visual Analytics
- Graphical visualization layer
- Multi-dimensional portfolio views:
  - IT domains
  - Budget breakdowns
  - Technology mix
  - Global delivery model
  - Risk analysis
- Dynamic, real-time analysis criteria changes
- Visualizations generated in seconds

### Cloud Migration Planning
- Hyperscaler pricing simulation (AWS, Azure, GCP)
- Database server instance pricing
- Application server instance pricing
- CO2 savings projections

## Assessment Deliverables

### Business Case Summary
- Current State Costs

### Portfolio Analysis Reports
- 360° IT application portfolio view

### Cloud Migration Roadmap
- Application-by-application recommendations
- Migration effort estimates

## Output Example: Application Disposition

For each application, eAPM provides:

| Field | Description |
|-------|-------------|
| Application Name | Identifier |
| Current State | On-premises infrastructure details |
| Cloud Readiness Score | 1-5 scale |
| Recommended Strategy | Rehost, Replatform, Refactor, etc. |
| Migration Effort | Low/Medium/High |
| Blockers | Technical or compliance barriers |
| Target Cloud | AWS, Azure, GCP recommendation |
| Estimated Cost | Current vs cloud TCO |
| CO2 Impact | Environmental savings |
