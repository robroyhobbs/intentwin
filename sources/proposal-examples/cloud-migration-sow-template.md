## 3. Scope of Work

### 3.1 In Scope

#### Phase 1: Assessment (X weeks)
- Application portfolio analysis using eAPM methodology
- Cloud readiness assessment for [X] applications
- Migration strategy development (6 Rs analysis)
- Business case and ROI modeling
- Migration roadmap and wave planning

**Deliverables:**
- Assessment Report
- Cloud Readiness Scorecard
- Migration Business Case
- Migration Roadmap

#### Phase 2: Foundation (X weeks)
- Cloud landing zone design and implementation
- Security baseline configuration
- Network and connectivity setup
- Identity and access management
- DevOps toolchain implementation

**Deliverables:**
- Landing Zone Design Document
- Security Baseline Documentation
- Network Architecture
- IAM Configuration
- CI/CD Pipeline Setup

#### Phase 3: Migration Waves (X weeks per wave)
- Wave [1-N] application migration
- Application testing and validation
- Cutover execution
- Hypercare support

**Deliverables per Wave:**
- Migration Runbook
- Test Results
- Cutover Report
- Operational Handover

### 3.2 Out of Scope
- [List specific exclusions]
- Application development or enhancement
- Data migration beyond specified systems
- Third-party license procurement
- Network hardware procurement

### 3.3 Client Responsibilities
- Provide access to systems and environments
- Assign client project manager and SMEs
- Timely decision-making and approvals
- Provide test data and test resources
- Procure required licenses
